# Data Analyst Job Market Analysis (2023)

This project analyzes over **32,000 data analyst job postings** to understand hiring trends, job demand, and application patterns.

The analysis was conducted using **Microsoft Excel**, including data cleaning, pivot tables, and dashboard visualization to transform raw job posting data into actionable insights.

## Key Questions

This analysis aims to answer the following questions:

- Which month recorded the highest and lowest number of job postings?
- Which data analyst roles are most in demand?
- Which cities hire the most data analysts?
- What proportion of job postings use Easy Apply versus standard applications?

## Data Cleaning Process

Before building the dashboard, the dataset was cleaned and prepared using Excel.

Key cleaning steps included:

- Converted categorical values from French to English (VRAI/FAUX → TRUE/FALSE)
- Standardized date formats
- Removed duplicate records
- Extracted month values from application dates
- Organized raw data into structured tables

## Analysis Process

After cleaning the dataset, exploratory analysis was conducted using pivot tables and Excel formulas.

The following analyses were performed:

- Monthly job posting trends
- Comparison between Easy Apply and standard applications
- Identification of the most in-demand Data Analyst roles
- Identification of cities with the highest hiring demand

## Dashboard Components

The Excel dashboard includes the following elements:

- KPI indicators (Total Jobs, Highest Month, Lowest Month, Average per Month)
- Monthly job posting trend visualization
- Easy Apply vs Standard Application comparison
- Most in-demand Data Analyst roles
- Top cities hiring Data Analysts

## Key Insights

- July recorded the highest number of job postings.
- February showed the lowest hiring activity.
- Business Data Analyst is the most in-demand role.
- New York leads hiring demand among major cities.
- Most job postings require standard applications rather than Easy Apply.

# Analyse du marché des Data Analysts (2023)

Ce projet analyse plus de **32 000 offres d'emploi de Data Analyst** afin d’identifier les tendances du marché, la demande par rôle et les méthodes de candidature.

L’analyse a été réalisée avec **Microsoft Excel**, en utilisant des techniques de nettoyage de données, des tableaux croisés dynamiques et un tableau de bord interactif.

- Microsoft Excel
- Tableaux croisés dynamiques

- Visualisation de données

## Questions analysées

- Quel mois présente le plus et le moins d’offres ?
- Quels rôles de Data Analyst sont les plus demandés ?
- Quelles villes recrutent le plus ?
- Quelle proportion des offres utilise Easy Apply ?

## Processus de nettoyage des données

Avant la création du tableau de bord, les données ont été nettoyées et préparées dans Excel.

Les principales étapes de nettoyage comprennent :

- Conversion des valeurs catégorielles du français vers l’anglais (VRAI/FAUX → TRUE/FALSE)
- Standardisation des formats de dates
- Suppression des doublons
- Extraction du mois à partir des dates de candidature
- Organisation des données brutes en tables structurées

## Processus d'analyse

Après le nettoyage des données, une analyse exploratoire a été réalisée à l’aide de tableaux croisés dynamiques et de formules Excel.

Les analyses suivantes ont été effectuées :

- Analyse des tendances mensuelles des offres d’emploi
- Comparaison entre les candidatures Easy Apply et les candidatures standard
- Identification des rôles de Data Analyst les plus demandés
- Identification des villes avec la plus forte demande de recrutement

## Composantes du dashboard

Le tableau de bord Excel comprend les éléments suivants :

- Indicateurs KPI (Total des offres, mois avec le plus d’offres, mois avec le moins d’offres, moyenne mensuelle)
- Visualisation des tendances mensuelles des offres d’emploi
- Comparaison entre Easy Apply et les candidatures standard
- Identification des rôles de Data Analyst les plus demandés
- Identification des villes recrutant le plus de Data Analysts

## Principaux résultats

- Le mois de **juillet** présente le plus grand nombre d’offres.
- **Février** est le mois avec la plus faible activité de recrutement.
- **Business Data Analyst** est le rôle le plus demandé.
- **New York** est la ville avec le plus d’opportunités.
